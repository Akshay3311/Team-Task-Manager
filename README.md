## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/team-task-managerr.git
cd team-task-managerr
```

### 2. Setup Backend

```bash
cd backend
npm install
npm start
```

### 3. Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
PORT=5000
DB_URL=your_database_url
JWT_SECRET=your_secret_key
```

---

## 📡 API Endpoints (Sample)

* POST /api/auth/signup
* POST /api/auth/login
* GET /api/projects
* POST /api/tasks
* PUT /api/tasks/:id
